Веб-приложение для подбора кулинарных рецептов по имеющимся продуктам

Основные дисциплины модуля: Технология разработки программного обеспечения и

Цель практики: определить тему дипломного проекта и выполнить первичные этапы проектирования и
разработки.

 
1.	Описание дипломного проекта



- Aктуальность темы.

  По данным опросов более 60% студентов хотя бы раз в неделю испытывают трудности с готовкой. Особую остроту проблеме придаёт тот факт, что многие студенты только начинают свою самостоятельную жизнь. Они впервые живут отдельно от родителей, учатся распоряжаться бюджетом, готовить для себя и пока не обладают достаточным кулинарным опытом, чтобы оценивать, какие рецепты им по силам. 

  Важно отметить, что целевая аудитория проекта не ограничивается студентами. Приложение будет полезно людям любого возраста, которые ценят своё время и не хотят тратить его на придумывание блюд.


- Целевая аудитория.

Исходя из актуальности темы, можно понять, что целевая аудитория разрабатываемого веб-приложения охватывает людей всех возрастов, однако основное ядро составляют пользователи от 18 до 30 лет - студенты и молодые специалисты.


- Предполагаемые функциональные возможности.

  У пользователя: просмотр списка продуктов, просмотр списка рецептов, просмотр карточки рецепта, выбор продуктов из холодильника (пользователь отмечает галочками продукты, которые у него есть), указание бюджета при поиске, сортировка результатов, избранное рецептов
  У админа: добавление/удаление рецепта, добавление/удаление/редактирование продукта, просмотр статистики



- Используемые технологии

Фронтенд: HTML, CSS, JavaScript, 
Бэкенд: Node.js, Express
База данных: MySQL



- Ожидаемый результат

В результате выполнения дипломного проекта будет создано работающее веб-приложение, позволяющее пользователю в несколько кликов найти блюдо из имеющихся продуктов с учётом бюджета на докупку. Программный продукт сопровождается пояснительной запиской, содержащей диаграммы, описание архитектуры, примеры SQL-запросов и руководство пользователя. Результат полностью соответствует поставленным задачам и может быть использован студентами и другими людьми для быстрого подбора рецептов без необходимости придумывать блюда самостоятельно.
